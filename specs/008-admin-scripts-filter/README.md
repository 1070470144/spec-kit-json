## 🎨 设计

### 筛选按钮组
```
┌─────────┬─────────┬─────────┬─────────┐
│ 待审核  │ 已通过  │ 已拒绝  │ 已废弃  │
│(默认)   │         │  (新增) │         │
└─────────┴─────────┴─────────┴─────────┘
```

### 状态映射
- **pending** → 待审核（默认）
- **published** → 已通过
- **rejected** → 已拒绝（新增）
- **abandoned** → 已废弃
